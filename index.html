EXPERIENCIA

PROFESIONAL

Desarrollador Freelance – Galería de Arte Online

Proyecto freelance · Febrero 2025 [Argentina] [Desarrollo Web]

Desarrollé una plataforma web completa para una galería de arte, permitiendo a los artistas exponer y gestionar sus obras en línea, y a los visitantes explorar el catálogo de manera visual e intuitiva.

  • Implementé el backend con Java Spring Boot, conectando con una base de datos PostgreSQL para gestionar artistas, obras, categorías y eventos.
  • Diseñé un frontend responsive en HTML, CSS y JavaScript, priorizando una experiencia de usuario limpia y elegante.
  • Incorporé autenticación para administradores y artistas, con funcionalidades CRUD para gestionar contenido de forma segura.
  • El proyecto está actualmente en línea y en uso, alojado en un servidor con Docker para facilitar el despliegue y mantenimiento.
